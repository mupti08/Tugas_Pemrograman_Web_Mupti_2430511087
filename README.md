# FIXTLAPBOT ID - Sistem Manajemen Servis Laptop Berbasis AI

FIXTLAPBOT ID adalah sebuah aplikasi berbasis web untuk mengelola administrasi dan pendataan servis laptop. Mengusung tema asisten robot AI, sistem ini dirancang untuk mencatat keluhan pelanggan, mendokumentasikan bukti fisik kerusakan secara mendetail, dan menyimpan persetujuan pelanggan secara digital dalam satu dashboard yang interaktif.

## Progres & Fitur yang Sudah Selesai (100%)

Sistem ini telah dilengkapi dengan berbagai fitur modern untuk menunjang efisiensi pendataan:

- Sistem Autentikasi (Login)
  Akses masuk aman menggunakan username dan password untuk melindungi data pelanggan dan operasional dashboard.
  
- Manajemen Data Servis (C-R-U-D)
  Kemampuan penuh untuk Membuat (Create), Membaca (Read), Memperbarui (Update), dan Menghapus (Delete) data nomor resi, nama pelanggan, merk laptop, keluhan, dan status pengerjaan secara real-time tanpa perlu reload halaman berkat teknologi AJAX.

- Upload Multiple File (Foto Bukti Fisik)
  Sistem dapat menerima dan menyimpan banyak foto sekaligus (tampak depan, nomor seri, detail kerusakan) dalam satu nomor resi servis.

- Tanda Tangan Digital (E-Signature)
  Terintegrasi dengan Canvas interaktif, memungkinkan pelanggan untuk langsung membubuhkan tanda tangan persetujuan secara digital melalui layar.

- Pencarian Data & Pelaporan (DataTables)
  Tabel data sudah terintegrasi dengan DataTables, menyediakan fitur pencarian cepat, penomoran halaman, dan ekspor laporan ke format Excel, PDF, dan Print.

- Antarmuka Interaktif (Bootstrap Modals)
  Formulir tambah dan edit data muncul dalam bentuk pop-up (Modal) yang rapi, memastikan pengguna tetap fokus pada halaman dashboard tanpa harus berpindah-pindah jendela.

- Video Banner Animasi (16:9)
  Halaman dashboard dilengkapi dengan pemutar video banner latar belakang yang berputar secara otomatis dan proporsional (rasio 16:9), memberikan kesan futuristik.
